# 托业错题本

给托业（TOEIC）错题做知识沉淀的仓库。丢一道做错或想收藏的题进来，AI 助手会拆解考点，把可复用的知识点归类记进 `notes/` 下的笔记，方便日后复习。

同时支持 **Claude Code** 和 **Codex**：核心指令写在 `AGENTS.md`（Codex 原生读取），`CLAUDE.md` 只用一行 `@AGENTS.md` 引入同一份指令，避免两份说明漂移。

## 怎么用

1. 打开 Claude Code 或 Codex。
2. 直接粘贴一道托业题（题干 + 选项，有正确答案或截图更好）。
3. 助手会拆题、归类、写入 `notes/`，最后用几行告诉你更新了哪些文件。
4. 复习时翻 `notes/` 下对应分类笔记；`01_错题本.md` 是总索引。

没粘题时助手不会主动改文件——可以正常向它提问，或让它帮你查笔记。

所有 `notes/` 笔记都采用追加模式：新内容写到对应文件末尾；已存在的条目只补充，不重复建条。

## 可视化查看

双击 `index.html` 用浏览器打开所有笔记：左侧切换文件、顶部搜索框跨笔记查词（命中处高亮）。单文件、离线、跨平台，不用装任何东西，也能发到手机看。

`index.html` 由 `notes/` 内容生成。笔记更新后让 AI 重新生成一次即可（或在装了 Python 的环境运行 `python3 build.py`）。

## 文件结构

| 文件 | 内容 |
|------|------|
| `AGENTS.md` | 核心工作指令（Codex 原生读，Claude Code 经 CLAUDE.md 引入） |
| `CLAUDE.md` | 仅 `@AGENTS.md` 一行 |
| `notes/01_错题本.md` | 完整题目存档 + 解析，索引源 |
| `notes/02_高频词汇.md` | 词性、释义、题中例句、易混词 |
| `notes/03_固定搭配.md` | 动词介词搭配、固定短语 |
| `notes/04_语法点.md` | Part 5/6 语法考点：规则 + 例证 |
| `notes/05_长难句.md` | Part 7 长难句结构拆解 + 翻译 |
| `notes/06_商务场景表达.md` | 邮件 / 会议 / 合同等场景表达 |
| `notes/07_易混词辨析.md` | 形近、义近词辨析 |
| `index.html` | 浏览器查看器（双击即开，自包含·离线），由下方脚本生成 |
| `build.py` | 从 `notes/` 重新生成 `index.html`（仅标准库） |

分类不够用时会新增文件，编号顺延并在此表登记。
