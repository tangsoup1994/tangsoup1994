# 07 易混词辨析

形近、义近词放一起辨析，讲清区别点（词义侧重、搭配、可数性、及物性……）。

| 易混词组 | 区别（侧重 / 搭配 / 用法） | 题中例句（英文原文） |
|----------|---------------------------|----------------------|
| rapid / bright / loud / proud | `rapid` 表速度快，能修饰 `expansion`；`bright` 表明亮、聪明或前景好，不表示扩张速度；`loud` 多指声音大或颜色醒目；`proud` 表自豪，通常修饰人或组织。 | has seen rapid expansion in East Asia |
| large / larger / largely / largest | `large` 是原级；`larger` 是比较级；`largest` 是最高级，和 `the ... of all` 搭配；`largely` 是副词，表示「主要地 / 很大程度上」，不能作最高级形容词。 | the largest of all the company’s retail locations |
| teach / taught / teaching / teachers | `teach` 是动词原形；`taught` 是过去式 / 过去分词；`teaching` 是动名词或现在分词，不表示复数人员；`teachers` 是复数名词，可作 `are` 的主语。 | Singh Design Institute’s teachers are active professionals |
| often / so / quite / too | `so` 可和 `that` 构成 `so + adj. + that` 结果结构；`often` 表频率；`quite` 只表示程度，不能和 `that` 呼应；`too` 常接 `to do` 表过度。 | so powerful that it can cut sheets of metal |
| special / specialization / specialize / specialist | `special` 是形容词；`specialization` 是抽象名词，表示专业化或专长；`specialize` 是动词；`specialist` 是表示人的名词，可作职位身份。 | financial specialist, Richard Bigsby |
| expertise / expert / expertly / experts | `expertise` 是不可数抽象名词，表示专业知识；`expert` 是单数可数名词，指专家；`expertly` 是副词；`experts` 是复数名词，不能和 `an` 搭配。 | an expert in climate science |
| establish / established / establishing / was established | `establish` 是动词原形；`established` 可作过去分词，表示被动后置定语；`establishing` 是现在分词，表示主动；`was established` 是完整谓语，不能放在已有谓语的句子中。 | a new environmental agency established to address the city’s pollution problem |
| designating / designated / designation / designate | `designated` 是过去分词形容词，表示「指定的」；`designating` 是现在分词，偏主动；`designation` 是名词；`designate` 是动词原形，不能直接修饰 `parking space`。 | designated parking space |
| whenever / through / including / once | `through` 是介词，可接时间段表示持续贯穿；`whenever` 和 `once` 是连词，后面要接句子；`including` 表包括，不表示持续时间。 | continue through the upcoming holiday season |
